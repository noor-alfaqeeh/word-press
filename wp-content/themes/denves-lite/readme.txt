== Description ==

Denves Lite is a clean and modern WordPress theme, ideal to manage newspapers, personal blogs or magazines website and any kind of websites. It provides feature like a post slider to display your latest posts with an optional featured link section, is also available an header widget area suitable to place ads banner. Denves Lite is fully optimized for mobile devices, easily customizable thanks to the WordPress Live Customizer section and is also WooCommerce ready.
